1. Data Collection

Collected genuine and forged handwritten signature samples for supervised learning.

2. Preprocessing

Images are resized, normalized, and augmented to improve generalization and robustness.

3. Model Training

A CNN (ResNet50-based) model is trained using TensorFlow/Keras to learn signature patterns.

4. Decision Logic

Confidence thresholds classify signatures as Accept, Review, or Reject to reduce false decisions.

5. Deployment

Flask backend serves predictions, frontend provides interactive verification.

About This Project

This project, "Smart Signature Verification System," was built for the 3rd Semester AI project at VIT Pune. It uses a Convolutional Neural Network (CNN) to analyze offline signatures.

Core Technology

  • Backend: Python (Flask)
  • AI Model: TensorFlow/Keras (using a CNN, ResNet-based)
  • Frontend: HTML, CSS, and JavaScript (with SignaturePad.js)
  • Key Feature: A Threshold-Based Decision System classifies signatures not just as pass/fail, but as Accept, Review, or Reject, which is more practical for real-world scenarios in banking and legal sectors.
