The system follows a structured, production-inspired AI pipeline designed for accurate and explainable signature verification.

  • Input Acquisition: Signature is captured either through live canvas drawing or uploaded as an image file (PNG/JPG).
  • Preprocessing: The image is resized, normalized, and cleaned to remove background noise and improve feature clarity.
  • Feature Extraction: A deep Convolutional Neural Network (CNN) extracts distinctive stroke and pattern features.
  • Decision Logic: Based on the confidence score, the system classifies the signature as Accept, Review, or Reject.

This project demonstrates a practical AI-based approach to signature verification for fraud detection and document security.

It is designed for real-world applications such as banking, legal document validation, and identity verification, where accuracy and consistency are critical.
